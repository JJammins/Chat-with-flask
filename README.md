# 중간 테스트 웹 채팅 페이지

## 주요기능
* 대화의 내용만 DB에 저장
* 반응형 웹 디자인
* 로그인 (회원가입 X)
* 친구목록
* 채팅목록
* 채팅창
* fontawesome 아이콘 사용

## 기술스택
* 프레임워크 : Flask
* 템플릿엔진 : Jinja
* 데이터베이스 : SQLite
* HTML / CSS
